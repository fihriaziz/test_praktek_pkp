File Category PHP Akan Menghasilkan Output seperti berikut

```php
Array
(
    [Electronics] => Array
        (
            [0] => Keyboard
            [1] => Mouse
        )

    [Fruits] => Array
        (
            [0] => Apple
        )

)
```

File Faktorial HTML, akan menampilkan result dari faktorial yang di input.
